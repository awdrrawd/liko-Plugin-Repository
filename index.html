📦 安裝方式Installation

兩種安裝方式：PCM 管理器一次管理所有插件，或前往 /Plugins 資料夾單獨安裝。 Two ways to install: use PCM to manage all plugins at once, or browse /Plugins to install individually.

🧩
使用 PCM 管理器Via PCM Manager
一次安裝所有插件，支援啟用/停用、自動載入、快取加速 Install all plugins at once — enable/disable, auto-load, cache acceleration
🔌
單獨安裝Install Individually
前往 /Plugins 資料夾，找到想要的插件直接安裝，無需 PCM Browse /Plugins folder and install only what you need, no PCM required

方式一 · PCM 安裝Option 1 · Install via PCM

🐵 腳本管理器（Tampermonkey / Violentmonkey）Script Manager

點擊下方按鈕，透過腳本管理器安裝 PCM：Click the button below to install PCM via your script manager:

👉 點此安裝 PCMInstall PCM Now

🔖 書籤安裝Bookmark Installation

建立一個新書籤，將網址欄貼上以下程式碼：Create a new bookmark and paste the following as the URL:

javascript:(function(){
  var s=document.createElement('script');
  s.src="https://awdrrawd.github.io/liko-Plugin-Repository/Plugins/Liko-Plugin_Collection_Manager.user.js?"+Date.now();
  s.type="text/javascript";
  s.crossOrigin="anonymous";
  document.head.appendChild(s);
})();

💻 瀏覽器控制台Browser Console

開啟 F12 開發者工具，在 Console 分頁貼上：Open F12 DevTools and paste into the Console tab:

import(`https://awdrrawd.github.io/liko-Plugin-Repository/Plugins/Liko-Plugin_Collection_Manager.user.js?v=${(Date.now()/10000).toFixed(0)}`);

方式二 · 單獨安裝Option 2 · Install Individually

前往 /Plugins 資料夾，透過腳本管理器安裝 .user.js，每個插件均可獨立使用。 Browse /Plugins, install any .user.js via Tampermonkey. Each plugin works standalone.
